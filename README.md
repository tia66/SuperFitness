# SuperFitness
Landing page per palestra locale ottimizzata per conversione clienti e vendita abbonamenti. Design moderno mobile-first con CTA WhatsApp, sezioni persuasive e focus su lead generation. Progetto HTML/CSS/JS pensato per trasformare visite in iscrizioni e richieste di prova gratuita.
A conversion-focused landing page built for a local fitness business in Ercolano (Italy), designed to turn visitors into leads and paying members through strategic UX, persuasive copywriting, and WhatsApp-first funnel design.
Project Overview
This project is not a simple gym website — it’s a lead generation system disguised as a landing page.
The main objective is to maximize:
Trial bookings
WhatsApp inquiries
Subscription conversions
through a structured conversion funnel.

 Key Features
High-conversion landing page architecture
Mobile-first responsive design
WhatsApp integration as primary CTA channel
Persuasive marketing structure (problem → solution → proof → offer)
Social proof sections (testimonials & transformations)
Performance-oriented lightweight frontend (no frameworks)
Smooth UX interactions (scroll animations, navigation behavior)
 UX / Conversion Strategy
The page is built using real-world conversion principles:
Clear value proposition above the fold
Emotional triggers (before/after transformation logic)
Trust building through reviews and results
Repeated CTA placement for maximum conversion rate
Reduced friction user journey (1-click WhatsApp contact)
Tech Stack
HTML5
CSS3 (custom, no frameworks)
Vanilla JavaScript
 Business Impact Goal
Designed to help local fitness businesses:
Increase inbound leads without paid complexity
Improve conversion rate from existing traffic
Turn online presence into a client acquisition tool
 About the Developer
Built as part of a freelance-focused portfolio showcasing real-world marketing-driven development, UX thinking, and conversion optimization skills.
